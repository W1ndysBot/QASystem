# QASystem

基于 NLP 和编辑距离算法的知识库问答系统

## 命令

qa-on 开启问答系统

qa-off 关闭问答系统

qa-add 添加问答

qa-rm 删除问答

qa-list 查看知识库问答对列表

qa-solo 对比两个句子的相似度
